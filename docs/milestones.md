# Milestones
Los [milestones](https://github.com/jlgallego99/OSTfind/milestones) representan cada uno de los estados en los que va a ir evolucionando la aplicación, conformando nuevos productos mínimos viables. Estos se construyen encima de los anteriores, por lo que no se avanzará con la aplicación hasta que un milestone no quede completo en su totalidad, es decir, todos sus issues se hayan completado.

- [Milestone 0: Estructuras de datos de obras y canciones](https://github.com/jlgallego99/OSTfind/milestone/5)
- [Milestone 1: Información y lógica de los usuarios](https://github.com/jlgallego99/OSTfind/milestone/4)
- [Milestone 2: Algoritmo del sistema de recomendaciones](https://github.com/jlgallego99/OSTfind/milestone/3)
- [Milestone 3: Recomendaciones personalizadas para cada usuario](https://github.com/jlgallego99/OSTfind/milestone/6)
- [Milestone 4: API que ofrezca la funcionalidad](https://github.com/jlgallego99/OSTfind/milestone/2)
- [Milestone 5: Aplicación desplegada en la nube](https://github.com/jlgallego99/OSTfind/milestone/1)
